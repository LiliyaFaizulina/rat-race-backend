# rat-race-backend

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the server.
Open [http://localhost:4000/api-docs](http://localhost:4000/api-docs) to view it`s documentation in your browser.

### `npm run start:dev`

Runs the server in the development mode.\
Open [http://localhost:4000/api-docs](http://localhost:4000/api-docs) to view it`s documentation in your browser.
